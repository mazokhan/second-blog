My journey into the coding world

This was a very interesting project where we were required to essentially find an API and be able to ask questions and respond to get information back.


I ended up finding a 'Dog Facts' API - Which was quite interesting and fun to do. I initially decided a movie database however the movie api only consisted data about songs!.
The whole part of this project was to find random information about Dogs.


The first question that comes up is 'How Many dog facts do you want?'. This is where you can choose how many you want and the limit is upto 100.


Then a question is asked 'Do you want them in random order?' And that is a simple Y/N option. The next question followed up is 'Do you want to know more facts?' - Which again is a yes or no answer. The Gemfile used for scraping is Mechanize. This is how i was able to get the information back from the API.
